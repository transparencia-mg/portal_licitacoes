# 📊 Portal de Licitações – Governo de Minas Gerais

Dashboard interativo para **visualização, consulta e download de dados públicos sobre licitações** realizadas pelos órgãos e entidades do Poder Executivo do Estado de Minas Gerais.

A solução organiza e disponibiliza dados públicos em formato aberto, permitindo análise, reutilização e acompanhamento das contratações públicas realizadas pelo governo.

---

## 🔍 Visão Geral

Este projeto disponibiliza um **dashboard web público** com informações sobre processos licitatórios do Estado de Minas Gerais.

O objetivo é ampliar a **transparência das compras públicas**, permitindo que cidadãos, pesquisadores, jornalistas e gestores públicos possam consultar e analisar os dados de forma simples e acessível.

### Principais funcionalidades

- Visualização de licitações por **órgão público**
- Consulta por **modalidade de licitação**
- Filtros por **situação da licitação**
- Pesquisa por **número do processo ou objeto**
- Visualização de **detalhes do procedimento licitatório**
- Download dos dados em formato aberto
- Atualização contínua a partir de dados estruturados

---

## 📑 Sobre os Dados de Licitações

Os dados disponibilizados no dashboard referem-se aos **procedimentos licitatórios realizados pelos órgãos e entidades do Poder Executivo do Estado de Minas Gerais**, incluindo informações como:

- Número do processo
- Órgão responsável
- Modalidade da licitação
- Situação do processo
- Objeto da contratação
- Datas relevantes do procedimento
- Informações complementares do processo

Esses dados permitem acompanhar de forma transparente **como o governo realiza suas compras e contratações públicas**.

A divulgação dessas informações atende aos princípios de transparência estabelecidos pela **Lei de Acesso à Informação (Lei nº 12.527/2011)**, que determina a publicação ativa de dados de interesse coletivo pelos órgãos públicos.

---

## 🗂️ Estrutura dos Dados

Os dados utilizados pelo dashboard são organizados em arquivos estruturados que permitem processamento automatizado e atualização contínua.

Essa estrutura facilita:

- organização das informações
- reutilização por outros sistemas
- integração com plataformas de dados abertos
- manutenção e versionamento das bases de dados

---

## 📦 Dados Abertos

Os dados deste projeto são disponibilizados em formato aberto, permitindo:

- reutilização por pesquisadores e jornalistas
- integração com outras plataformas
- análise automatizada
- criação de aplicações baseadas em dados públicos

Os conjuntos seguem padrões utilizados em portais de dados abertos, possibilitando sua publicação em plataformas como **CKAN** e outros catálogos de dados governamentais.

---


Este projeto integra as iniciativas de **transparência ativa do Governo do Estado de Minas Gerais**, ampliando o acesso às informações públicas e fortalecendo o controle social sobre as ações governamentais.

O objetivo é facilitar a consulta e promover maior compreensão sobre **os processos de contratação pública realizados pelo Estado**.
